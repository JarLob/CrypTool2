PKCS#1 signature generation

PKCS#1 signatures are based on the asymmetric encryption algorithm RSA. Therefore we need to generate an RSA key.
In order to generate a PKCS#1 signature we first create a data block and a padding string, which helps to format the signature in a specific way. The resulting data block is then signed with the private RSA key.

Please notice that you first have to generate an RSA key pair and a data block before you can generate the PKCS#1 signature!
