Number-theoretic functions

Number-theoretic (or arithmetic) functions are functions whose domain is the set of natural numbers and that take real or complex values. In number theory, the properties of natural numbers are examined and described, especially with respect to their divisibility.

A special type of number-theoretic functions are the multiplicative number-theoretic functions. They can be evaluated easily by using the unique factorization of a number:
if n1 and n2 are coprime natural numbers, then f(n1*n2) = f(n1) * f(n2).

In this presentation you can examine the following functions:
  • Number of primes: calculates the number of prime numbers <= n
  • Euler's totient function: calculates how many numbers <= n have no common divisor with n
  • List of coprimes: displays all numbers < n that are coprime to n
  • Number of divisors: calculates the number of divisors of n
  • List of divisors: displays all divisors of n
  • Sum of divisors: calculates the sum of all divisors of n
  • Sum of divisor totients: sums Euler's totient function for all divisors of n. This must add up to n.
Some functions need a second parameter m besides the control variable n, that must be entered in the respective textfield. The meaning of the second parameter depends on the function and is explained below:
  • Greatest common divisor: calculates the gcd of n and m
  • Least common multiple: calculates the lcm of n and m
  • Extended Euclidean algorithm: calculates the coefficients a and b that are necessary to represent gcd(n,m) as a linear combination of n and m, i.e. a*n + b*m = gcd(n,m)
  • Modular multiplicative inverse: calculates the multiplicative inverse of n in the residue class ring Zm, provided it exists (i.e. gcd(n,m)=1)
