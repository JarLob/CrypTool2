Number-theoretic functions

Number-theoretic (or arithmetic) functions are functions whose domain is the set of natural numbers and that take real or complex values. In number theory, an number theory, they are used to examine and describe the properties of natural numbers, especially with respect to their divisibility.

A special type of number-theoretic functions are the multiplicative number-theoretic functions. They can be evaluated easily by using the unique factorization of a number:
if n1 and n2 are coprime natural numbers, then f(n1*n2) = f(n1) * f(n2).

Some examples of number-theoretic functions are:
  • Euler's totient function
  • the Möbius function
  • the number of divisors function
  • the sum of divisors function
