Euler's φ-Function

Euler's φ-function is an arithmetic function. For any natural number n, the function returns a number which is the count of integers between 1 and n which are relatively prime to n.

The name of the function goes back to Leonhard Euler who first introduced it in the Fermat-Euler Theorem which is a generalization of Fermat's Little Theorem.
The following apply:
  • φ (n) <= n-1 for all n ∈ N and
  • φ (n) = n-1 ⇔ n is prime.

One can therefore say that the smaller φ (n) is, compared to n-1, the more divisors n will have. Conversely, φ (n) is considered to be an indicator of how likely it is that the natural number is prime. This likelihood is useful in primality testing, although the final determination deals with the yes/no characteristic of primality.

Therefore φ (n) = n-1, if n is a prime number.

φ (n) is one of the most important number-theoretic functions. The more we learn about it, the more we know about prime numbers and their distribution. Since n ∈ N is prime if and only if φ (n) = n-1, it follows that knowing all solutions to the equation φ (n) = n-1 is equivalent to knowing the sequence of prime numbers.
