Pari/GP for the evaluation of the logarithmic integral

Pari/Gp is an Open Source Computer Algebra System (CAS), that offers very fast implementations of arithmetic functions.
It evaluates the logarithmic integral with much more precision than using Simpson's rule.
Pari/Gp can be downloaded from the webite of the University of Bordeaux.
Activate the checkbox "Pari/GP" for using Pari for the evaluation of the logarithmic integral and select the program "gp.exe" in the file dialog. The program is in the Pari/GP installation folder.
